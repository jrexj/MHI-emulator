# MHI Emulator

## Overview
The MHI Emulator is designed to simulate the functionality of the MHI system, enabling developers and users to test and validate their applications without accessing the actual MHI system.

## Features
- **Simulation of MHI System**: Mimics the behavior and responses of the MHI system.
- **User-Friendly Interface**: Easy-to-use interface for testing.
- **Logging**: Detailed logging of all interactions for troubleshooting purposes.

## Setup Instructions
1. **Clone the Repository**:  
   Run the following command to clone the repository:
   ```
   git clone https://github.com/jrexj/MHI-emulator.git
   ```

2. **Navigate to the Directory**:  
   Change into the project directory:
   ```
   cd MHI-emulator
   ```

3. **Install Dependencies**:  
   Use the package manager of your choice to install the necessary dependencies:
   - For npm: 
   ```
   npm install
   ```
   - For pip:
   ```
   pip install -r requirements.txt
   ```

4. **Configuration**:  
   Update the configuration file `config.json` with your specific settings.

5. **Running the Emulator**:  
   To start the emulator, use the following command:
   ```
   npm start
   ```
   or
   ```
   python main.py
   ```

6. **Accessing the Interface**:  
   Open your web browser and go to `http://localhost:3000` to access the user interface.
